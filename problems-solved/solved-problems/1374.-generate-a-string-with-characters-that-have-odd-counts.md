# 1374. Generate a String With Characters That Have Odd Counts

```cpp
class Solution {
public:
    string generateTheString(int n) {
      
      string s = "";
      
      if((n&1)==1)
      {
        for(int i=0;i<n;i++)
          s+='a';
      }else
      {
        for(int i=0;i<n-1;i++)
          s+='a';
        s+='b';
      }
      
      return s;
        
    }
};
```
