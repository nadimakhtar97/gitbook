# 1518. Water Bottles

```cpp
class Solution {
public:
    int numWaterBottles(int n, int k) {
      
      
      int res = n;
      int empty = n;
      
      while(empty/k)
      {
        res += empty/k;
        empty = empty%k + empty/k;
        
      }
      
     
      return res;
        
    }
};
```
